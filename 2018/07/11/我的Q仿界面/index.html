我的Q仿界面

发表于 2018-07-11 | 更新于 2018-07-23 | 分类于 Java

这是大学期间做的，转眼已经过了好多年了，在此纪念一下。

当时是看到Java界面布局(Swing)，就拿QQ来练手，做了登陆界面和聊天界面。由于主要是界面，大部分功能键只添加了触发状态和显示效果，没有实际功能。作为一个强迫症，当时调界面尺寸是精细到像素级别，想想也是可怕…

另外当时也是第一次上传优酷视频，最奇葩的是本来录制的很高清的视频上传后变得很模糊，然后我调整码率，降低分辨率后重新上传，结果就没被压缩，视频依旧很清晰。只想对优酷的压缩算法表示呵呵。

当然，Java桌面应用开发也就玩玩而已，在公司中就再也没用过啦。
